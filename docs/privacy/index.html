Privacy Policy

Last Updated: January 15, 2026

AtlaSynq Inc. ("we," "our," or "us") provides a unified intelligence layer for enterprise data (the "Service"). We are committed to protecting the privacy and security of your proprietary data. This Privacy Policy describes how we collect, use, and disclose information.

By using our Service, you agree to the collection and use of information in accordance with this policy. This policy applies to our website, platform, and all related services.

2. Data Collection & Usage

2.1 Customer Data

We process data from your integrated tools (e.g., Salesforce, Slack, Jira) solely to provide answers to your queries. This data is processed in real-time and is typically not stored persistently unless required for specific features enabled by you.

2.2 Usage Data

We collect anonymized telemetry data regarding how users interact with the platform (e.g., number of queries, response latency) to improve system performance and reliability.

3. AI & Data Governance

Strict Zero-Training Policy: We do not train, fine-tune, or improve our foundation models using your proprietary data. Your business data remains yours.

When you ask a question via AtlaSynq:

  • Relevant context is retrieved securely from your connected tools via API.
  • This context is sent to the Large Language Model (LLM) provider via secure API channels.
  • We operate under zero-retention agreements where possible, minimizing the time your data exists on external servers.

4. Security Measures

We take security seriously and implement industry-standard measures to protect your information.

  • Encryption: Data is encrypted in transit using TLS 1.3 and at rest using standard security protocols.
  • Access Control: Our system respects the permissions of the source platform. If a user does not have access to a file in the source system (e.g., Jira), the AI agent will not access it on their behalf.
  • Cloud Infrastructure: Our services are hosted on top-tier cloud providers (such as AWS/Vercel) that maintain strict physical and digital security standards.

5. Data Sharing

We do not sell your personal data. We share data only with trusted third-party service providers necessary to operate the service (e.g., for hosting, analytics, or LLM processing), and we require them to maintain confidentiality.
