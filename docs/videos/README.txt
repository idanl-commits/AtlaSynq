AtlaSynq Product Tour Video
============================

Drop your product tour video here as:

  atlasynq-product-tour.mp4

The homepage at /index.html references this file at:

  /videos/atlasynq-product-tour.mp4

Recommended specs:
  - Format: H.264 MP4 (broadest browser support)
  - Resolution: 1280x720 (720p) or 1920x1080 (1080p)
  - Duration: 2–4 minutes recommended
  - Max file size: 80 MB (serve via CDN for larger files)

When no video file is present, the site automatically shows
an interactive 4-step fallback (Connect → Assign role →
Ask with citations → Admin approval) that functions as a
product walkthrough until the video is uploaded.
