How governance actually works

Pick the model that fits the department. Keep one policy layer for tools, citations, and write approvals — regardless of whether the answer is from GPT, Claude, Gemini, or Llama.

01

Assign model per role

Sales on GPT-4o, Finance on Claude, Engineering on Gemini or self-hosted Llama. Switch providers without rebuilding connectors.

02

Enforce RBAC before the model

Tool scope is checked first. If Finance cannot see Salesforce, that context never reaches any LLM.

03

Cite, then approve writes

Answers link to source records. CRM updates and outbound messages wait in an admin queue — model-agnostic audit trail.
